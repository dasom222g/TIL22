### Context
n8n을 활용하여 Telegram 메시지를 자동화하는 작업을 진행했습니다. 이번 자동화의 목적은 특정 이벤트가 발생할 때마다 Telegram을 통해 알림을 받는 것입니다. 이를 통해 중요한 정보를 즉시 확인하고 대응할 수 있습니다.

### Core
Telegram과 n8n의 통합을 설정하기 위해 다음과 같은 과정을 따랐습니다.

1. **Telegram 봇 생성**
   * Telegram에서 BotFather를 통해 새 봇을 생성하고 토큰을 발급받습니다.
   
2. **n8n에서 Telegram 노드 추가**
   * n8n 워크플로우에 Telegram 노드를 추가하고, 발급받은 토큰으로 인증을 설정합니다.
   
3. **자동화 트리거 설정**
   * n8n의 다른 노드와 연결하여 트리거 설정을 완료합니다.
   * 예를 들어, 새로운 데이터가 저장될 때마다 Telegram으로 알림을 보내는 방식으로 워크플로우를 구성할 수 있습니다.

```javascript
{
  "nodes": [
    {
      "name": "Telegram",
      "type": "telegram-send-message",
      "credentials": "Telegram Api",
      "parameters": {
        "text": "새로운 이벤트가 발생했습니다!"
      }
    }
  ]
}
```

### Insight
이 설정을 통해 특정 이벤트가 발생할 때마다 Telegram을 통해 실시간으로 알림을 받을 수 있습니다. 이는 특히 빠른 피드백이 필요한 환경에서 유용하며, 다양한 트리거와 결합하여 더욱 복잡한 자동화 시나리오 구축이 가능합니다.
