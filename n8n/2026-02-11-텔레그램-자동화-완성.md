### Context
n8n을 사용하여 Telegram과 다른 여러 애플리케이션 간의 자동화를 구현하고자 합니다. Telegram은 강력한 커뮤니케이션 도구로, 이를 n8n과의 통합을 통해 다양한 자동화 시나리오를 구축할 수 있습니다. 여기서는 Telegram API를 사용하여 메시지 자동화를 설정하는 방법을 이해하고자 합니다.

### Core
n8n에서 Telegram 자동화를 설정하는 기본 단계는 다음과 같습니다:

1. **Telegram Bot 생성**: `@BotFather`를 통해 새로운 봇을 생성하고 API 토큰을 얻습니다.
2. **n8n 설정**: n8n 워크플로우에서 **Telegram 노드**를 추가합니다. "Send Message" 액션을 선택하고, 얻은 API 토큰을 사용하여 인증을 설정합니다.
3. **트리거 설정**: 예를 들어, 특정 이벤트 발생 시 메시지를 보낼 수 있도록 **Webhook 트리거**를 설정합니다.

```json
{
  "nodes": [
    {
      "parameters": {
        "authentication": "telegramApi",
        "chatId": "<CHAT_ID>",
        "text": "Hello from n8n!"
      },
      "id": "Telegram_Send",
      "name": "Telegram",
      "type": "n8n-nodes-base.telegram",
      "typeVersion": 1,
      "position": [150, 300]
    }
  ]
}
```

### Insight
본 설정을 통해 Telegram 봇을 활용한 기본적인 메시지 전송 자동화를 구성할 수 있습니다. n8n의 유연성 덕분에 다양한 상황에 맞게 자동화를 커스터마이즈할 수 있습니다. 예를 들어, 데이터베이스나 다른 API와 연동하여 조건에 따라 메시지를 다르게 보낼 수 있습니다. 이러한 통합은 업무 프로세스를 크게 효율화할 수 있습니다.
